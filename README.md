# NEK 400 Electrical Planner MVP

Vite + React + Tailwind project for the NEK 400 Electrical Planner.

## Getting started

```bash
npm install
npm run dev
```

Then open the printed localhost URL in your browser.

## Deploying to Vercel

1. Push this folder to a new GitHub repository.
2. In Vercel, create a new project and import that repo.
3. Framework preset: **Vite** (or let Vercel auto-detect).
4. Build command: `vite build`
5. Output directory: `dist`
